SuburbMates — Canvas Export

This ZIP contains the four key React files created in Canvas:

1) Suburbmates-Directory.jsx — the first premium landing + layout.
2) Lead-Capture-Enhancements.jsx — the claim/register cards section.
3) SuburbMates-Directory-with-Leads.jsx — homepage with lead cards integrated + Pricing tab.
4) SuburbMates-Full-Package.jsx — polished package with premium typography and motion.

How to use:
- These are React components. Copy the JSX into your Next.js/React project as pages or sections.
- For static preview, wrap any component in a bare index.html with Tailwind + Framer Motion CDN.
- Replace links and routing with your framework's router (Next.js Link, etc.).
